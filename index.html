From action to action

This is a list of Action movies made by Anders Hultqvist showing his theory on collapsing visual signifiers, a way to contextualize action cinema.
It's also my second laboration with the Javascript library D3js. My goal for this example was to get out the movie-data with it's underlying connections. And also to bind click-events to the nodes for interaction and a way to display the movies complete information.
